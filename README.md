# Lalas - Sitio one-page

Propuestas HTML/CSS para la marca Lalas.

## Opciones

- `opcion-a-institucional.html`: enfoque institucional y comercial.
- `opcion-b-producto.html`: enfoque producto, variantes y lectura retail.
- `index.html`: selector de ambas opciones.

## Uso

Abrir `index.html` en el navegador o publicar la carpeta completa en GitHub Pages.

Todos los assets incluidos en `assets/` corresponden a materiales provistos para el proyecto.
